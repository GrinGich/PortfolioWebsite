Cosmetic customizer

Genre: Tool

Engine: Unreal Engine 5

Editor: Angelscript

Platform: PC

Time: 4 weeks


This project is aimed at making it easier to customize already existing cosmetics for purpose of creating variants of the same mesh.

Scene: Christopher Lundkvist
Mesh: a.moreno ( Fab )

The tool uses a recursive material to load patterns onto a material thus stacking patterns without the use of decals. The purpose of this tool was to independantly of any project; be able to load in any mesh and/or cosmetic and apply colour,patterns and emblems to the mesh and then exporting directly to the computer.

Inspired by Embark Studios in-house Cosmetic Customizer tool. It being the reason I challenged myself to Unreal Angelscript

  • Independent of any project the tool uses only patterns, meshes and textures from the users device making it to distribute.


  • Freedom from constraints of just cosmetics, anyhting with a singular texture would work with the customizer. Even a car!


  • Manipulate the patterns with multiple different modifiers, such as position, rotation, scale and colour.
  • Angelscript was used for making all the widgets used in the project.
    While still keeping to Unreals UI style.


  • Widgets consist of two main widgets. The first being the mesh settings where the texture, Normals and ORM.
    The Second widgets represents the Pattern and thus have all settings for the pattern, such as Position, Rotation, Scale and colour.


  • Save without interference from normals maps and ORM values to keep existing setups solid.
